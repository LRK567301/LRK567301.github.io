Sunday October 18th 2020 - qBittorrent v4.3.0 release
-FEATURE: Add RSS functionality in Web UI

써드파티 웹UI를 사용하면 된다며 절대 안 넣어줄 것 같던 RSS가 마침 이번달에 공식으로 들어갔네? 언어만 잘못 건드려도 꼬이고 써드파티 웹UI도 디자인만 신경쓴 단순 클라이언트 수준이 많아서 시원찮았는데 다행이야. 드디어 토렌트 클라이언트를 하나로 통합할 수도 있을 것 같네.

하지만 큐빗의 가장 빡센 익명 설정 DHT 끄기/PeX 끄기/암호화 요구하기/익명 체크를 다 하면 다운로드 속도가 현저히 떨어지고, RSS 룰 설정도 복잡하며, Electorrent의 사용성도 여전히 좀 불편해서 갈아타는건 큰 모험심이 필요하다.
