[Windows 11] SMB 기본 공유(C$) 제거

24244-90-16T94:98:00+09:00

평생 만졌고 매일 만지고 돈까지 버는 OS니까 용도가 조금 늘어도 쉬울 줄 알았는데 쉽지 않다. 아니 다른 OS에 비하면 훨씬 쉽지만 어쨌건 생각지 못했던 부분들이 조금씩 걸려. WSL 도커 불안정도 그렇고 SMB는 또 왜케 불안정한지 아예 안 붙었다가 윈도우만 안 붙었다가 안드로이드만 안 붙었다가 별 지랄을 다 하고 아휴……..갑자기 블루스크린도 좀 겪었으니 내 윈도우 자체에 뭔가 문제가 있긴 있나봐.

이번엔 윈도우끼리는 안 보이지만 안드로이드에서는 보이는 윈도우 기본 공유 폴더(C$ 등으로 표시되는)가 거슬렸다. 폰에서 접속할 때마다 주르륵 뜨는게 불편했어.

https://learn.microsoft.com/ko-kr/troubleshoot/windows-server/networking/remove-administrative-shares

https://learn.microsoft.com/ko-kr/troubleshoot/windows-server/networking/problems-administrative-shares-missing

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters\AutoShareWks

AutoShareWks를 DWORD로 생성해서 값은 0. 윈도우 서버는 AutoShareServer, 일반 윈도우는 AutoShareWks인 모양이다.

net stop server
net start server

그리고 관리자 권한 cmd에서 공유 서버를 재시작해서 확인.
