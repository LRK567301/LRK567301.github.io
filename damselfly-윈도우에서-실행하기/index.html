Damselfly 윈도우에서 실행하기

19196-30-06T36:38:00+09:00

WSL과 도커 걷어낼수 있을까의 일환.

먼저 Windows용 .NET 9.0을 설치하고 깃헙 저장소를 통째로 받는다. 다 필요해.

appsettings.json

{
  "ConnectionStrings": {
    "DefaultConnection": "DataSource=app.db"
  },
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "IdentityServer": {
    "Clients": {
      "Damselfly.Web.Client": {
        "Profile": "IdentityServerSPA"
      }
    }
  },
  "AllowedHosts": "*",
  "Kestrel": {
    "EndPoints": {
      "Http": {
        "Url": "http://0.0.0.0:6363"
      }
    }
  }
}

그리고 appsettings.json에 Kestrel로 외부 접속 가능하게 설정 추가.

Damselfly.Web.Server
dotnet run /path/to/my/photos

Damselfly.Web.Server 폴더에 들어가서 스캔할 폴더 지정하여 실행. 동영상이라던가 태그까지 가면 모르겠지만 이 최소한만으로 잘 실행되고 잘 스캔한다. 썸네일 표시 성능은 도커보다 좀 빠른 것 같기도 하고 아닌 것 같기도 하고.
