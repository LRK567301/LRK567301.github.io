갤S9에서 노트20, 압도적 차이에 감탄하며 몇주 쓰다 보니 뭔가 없어진 게 느껴진다. 알림LED가 없어. 소리나 진동이나 화면켜짐으로 시끄럽게 굴지 않으면서 시선이 살짝 가기만 하면 명확한 존재감을 뽐내던 LED가 없어졌어. AOD가 그 대신이겠지만 폰에서는 화면 항상 켜져 있는게 찜찜해서, 워치에서는 배터리 닳는게 느껴져서 써 본 적이 없다. 잠깐 시계 보는 용도로 터치시 10초면 충분했어.

근데 LED가 아예 없으니 안 되겠네. 손목에서 알려주기도 하지만 항상 시계를 차고 있는 건 아니다. 순정으로 끝내고 싶어서 잠깐 리마인더로 대체해봤지만 시도때도 없이 소리를 울려대는 건 짜증나. 조용히 화면으로만, 내 시선이 갔을 때만 알 수 있었음 좋겠어.

aodNotify가 답이었다. 터치시 AOD 10초 포기하고 링라이트니 뭐니 번잡한 거 다 끄고 알림 왔을 때만 AOD켜짐, 배터리 최적화 제외. 이틀쯤 쓰다 반응이 없어져서 왜 이러나 했더니 배터리 최적화 탓이었어.

VMware Player는 초기 설정 과정이 뭔가 매끄럽지 않았던 기억에, 그리고 스냅샷은 유료라 VirtualBox만 오래 써왔는데 오랜만에 설치해보니 성능 차이가 어마어마하다. 버박 윈도우가 너무 느려서 8.1, 심지어 7으로 돌아갈까 고민하다 포기하고 썼는데 VMware는 인증도 안한 윈도우10이 이렇게 매끈하네? 하………뭐하러 버박으로 버벅이며 고통받았지……………

하지만 여전히 초기 과정은 매끄럽지 않다. VMware의 프리셋을 따라 윈도우10을 자동으로 설치해버리면 VMWare tools 설치가 안 되고 재설치 메뉴도 비활성화 상태.
