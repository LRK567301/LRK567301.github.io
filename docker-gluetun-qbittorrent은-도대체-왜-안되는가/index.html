[Docker] gluetun+qbittorrent은 도대체 왜 안되는가

202010-110-22T1110:118:00+09:00

어느 레딧 투표에서 큐빗이 트랜스미션을 확 앞서는걸 보고 또 뽐이 왔다. 찾아보니 딱 1년전쯤 일주일쯤 잘 버티다 자살하길래 던졌던 기록이 있는데 이번엔 또 모르잖아?

gluetun 아래 한 묶음으로 잘 구성한 것 같은데 이번에도 역시 아니 더 심해서 시작조차 안 된다. 큐빗 웹UI의 RSS 리더가 주소 안 가리고 잘 갱신되는 걸 보면 gluetun 통해서 통신은 잘 된다는건데 정작 다운로드 걸면 트래커가 timed out, Operation not permitted, Host not found. 혹시나 해서 다운로드 경로도 아예 새 폴더로 바꿔보았지만 마찬가지.

됐다. 다신 안써.
