시놀로지 lqxt 패키지 설정

2020, 4월 30

버전 0.10.0-0012 기준. 도커 허브에서 온갖 이미지를 찾아 돌아다녔지만 시놀에선 시놀이 제공하는게 제일 깔끔하고 확실하고 편하다. 여기서까지 시놀 SW빨을 느껴.

기본적인 계정 암호는 synology. 패스워드 변경 바로가기를 실행하면 표시되기도 하지만 터미널부터 들어가면 고생한다. VNC 접속 암호도 꼭 지정해주자. 기본값은 그냥 열려 있으니까.

다음으로 너무 작은 해상도.

sudo vi /etc/service/xvfb/run

1680x1050으로 바꾸고 패키지 재시작. 상단 표시줄이 있다 보니 그 이상 해상도는 브라우저 창을 전체화면으로 해도 스크롤이 생긴다.

한국어 입력은

sudo apt-get install fcitx-hangul

시스템 도구에서 Fcitx를 실행해 트레이에 위치시키고 설정 Input Method에 Hagul, Keyboard - English (US)를 추가한다. Ctrl+Space로 한영전환, 텍스트 같은 기본앱에선 안되지만 크롬에선 잘 돼.

브라우징이 그렇게 안정적이진 않다. 기본 설치된 아주 구버전 크롬에서 아마존이나 네이버처럼 복잡한 페이지는 로드하고 잘 되는 듯 하다 죽어. 하지만 그 이하로 간단한 페이지는 꽤 안정적이고 빠르다. 확장 설치도 되고.
