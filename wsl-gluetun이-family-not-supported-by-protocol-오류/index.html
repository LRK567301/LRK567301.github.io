[WSL] Gluetun이 Family Not Supported By Protocol 오류

17174-40-16T44:48:00+09:00

도커 데스크탑 업데이트하더니 docker-compose 실행할 때 못 보던 버전 오류 메세지도 생겼길래 도커 탓인가 했는데 아니었다.

ERROR adding local rules: adding rule: fe80::/64: adding rule ip rule 98: from all to fe80::/64 table 254: address family not supported by protocol

https://github.com/qdm12/gluetun/issues/2213

WSL 버전 문제. 내가 아직 2.0.XX대를 쓰고 있더라. 프리릴리스까지 갈 것도 없이 wsl –update만으로 올라가는 최신 정식 2.1.5만으로 바로 해결. 도커 몇번 터지고 업데이트에 대해서 조금 보수적이 될 뻔 했는데 최신기술은 최신으로 따라가는게 맞나봐.
