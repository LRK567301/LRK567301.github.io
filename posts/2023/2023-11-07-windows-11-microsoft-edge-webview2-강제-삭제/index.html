[Windows 11] Microsoft Edge WebView2 강제 삭제

는 불가능하니 Revo Uninstaller로 직접 레지/파일을 지워주자.

Windows 11 23H2, 엣지 119.0.2151.44, 엣지 웹뷰2도 119.0.2151.44.

드디어 23H2가 나와서 재설치한건 좋았는데 아웃룩(New!) 같은게 좀 괜찮을까 싶어 켜봤다가 여전히 쓰레기임만 확인하고 지웠는데 Microsoft Edge WebView2가 자기 멋대로 설치되선 삭제되지도 않는다.

C:\Program Files (x86)\Microsoft\EdgeWebView\Application\119.0.2151.44\Installer
setup.exe --uninstall --msedgewebview --system-level --verbose-logging --force-uninstall

검색하면 나오는 인스톨 경로에 들어가서 삭제 명령도 전혀 반응이 없어. 삭제버튼이 눌리긴 하는 UninstallView 같은 외부 유틸리티로도 마찬가지.

포기하고 쓸까도 싶었지만 바로 옆에 아웃룩이나 오피스365 실행 안한 갤북은 너무 깔끔한 걸 보니, 그리고 엣지만 쓸 때는 웹뷰2 같은게 필요도 없는 걸 보니 짜증이 나서 더 찾아봤더니 관련글에서 Revo Uninstaller가 보인다. 맞아 그런 것도 있었지?

포터블 프리버전으로 진행해보니 얘도 인스톨러로 삭제해주는 건 아니고 레지랑 파일을 뒤져서 관련파일을 싹 지워주는 방식. 그래도 나쁘지 않다. 거슬리던 게 사라졌어.

+덧 아………이렇게만 지워서는 안되는구나 거의 쳐다보진 않지만 시작메뉴 왼쪽 위젯이 아무런 메세지 없이 동작하지 않는다. 그렇다고 웹뷰2를 받아서 설치하려고 해도 이미 설치되어 있다고 안 돼.

https://superuser.com/questions/1751709/how-to-reinstall-microsoft-edge-webview2-runtime

HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\EdgeUpdate\Clients\{F3017226-FE2A-4295-8BDF-00C3A9A7E4C5}

Revo가 완전히 다 지워준 것도 아니더라. 저 흔적까지 지워준 다음에야 웹뷰2를 다시 설치할 수 있었어. 그렇게 설치하니 제어판에서 삭제 항목이 활성화됐고, 그걸로 다시 삭제하니까 이제 시작메뉴 위젯이 웹뷰2가 필요하다는 오류를 띄운다. 너임마 웹뷰2 설치 전에는 없이도 됐던 놈이 ㅋㅋ 한번 설치되버리면 되돌릴 수 없구나.

+덧2 바빠서 일주일은 쳐다도 못 봤던 갤북에도 웹뷰2가 자동 설치된걸 발견했다. 애초에 잘못된 생각이었던 것.
