[Docker] docker-transmission-openvpn 웹 프록시 연결이 되지 않는다

컨테이너를 오래 굴리니까 또 시놀 도커 UI에서는 로그가 보이질 않길래 직접 docker logs로 확인해보니 멀쩡하게 Privoxy:Starting. 하긴 오늘 시놀이 드라이브 설정이 어쩌구 하면서 재부팅하래서 재부팅했을 뿐 어제까지는 잘만 쓴 컨테이너 그대로인데 달라진 게 있을 리가 없지.

https://github.com/haugene/docker-transmission-openvpn/issues/2489

혹시나 해서 검색해보니 도커 컨테이너 IP가 바뀌어서 문제인 모양. 컨테이너 재시작만으론 안 됐고 시놀 도커 UI에서 작업-재설정 눌러서 컨테이너 깨끗하게 새로 시작하니까 해결됐다. 트랜스미션-플렉스겟 처음 엮을 때 뭣도 모르고 내부 IP 열심히 입력하던 시절 이후론 포트만 열어주면 다 됐는데 오랜만에 부딪혔어.
