[VS Code] 날짜 및 시간 입력 스니펫

https://code.visualstudio.com/docs/editor/userdefinedsnippets

파일-기본 설정-사용자 코드 조각 구성

{
	"today": {
		"prefix": "today",
		"body": "$CURRENT_YEAR-$CURRENT_MONTH-$CURRENT_DATE T$CURRENT_HOUR:$CURRENT_MINUTE:00.000+09:00"
	}
}

Dendron 같은 본격적 메모 확장은 과한데 new dendron 하면 자동으로 양식 만들어주는건 부럽더라. 블로그용 양식 다 때려박으려다 막혀서 제일 귀찮았던 현재 날짜시간 입력 부분만.

의외의 관문은 메뉴 찾기였어. 한국어 메뉴에서 ‘사용자 코드 조각 구성’이다. 입력할 때는 Ctrl+Space. hugo의 시간 양식이 중간에 T가 꼭 붙어 들어가야 되는게 문제인데 스니펫 두번이나 치는 것보단 T 앞에 공백 한번 지우는게 빨라.
