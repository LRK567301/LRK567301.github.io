Nextcloud ONLYOFFICE 연동

ocis도 폴더 구조 때문에 보내주고 filestash도 모바일 사용성이 좀 그래서 기본값으로 같이 깔아주는 ONLYOFFICE 살릴 생각이 안 들어서 결국 다시 넥스트클라우드. 사진은 옆동네에 넘겨주고 노트앱이랑 웹 오피스만 해도 밥값하지 않나 싶다. Kasm 같은 걸로 원드라이브나 구글 오피스 빌려올까도 싶었지만 1얼마나 접속한다고 2모바일 사용성 둘 생각에 접었어.

https://github.com/ONLYOFFICE/onlyoffice-nextcloud

넥스트클라우드 앱, 넥스트클라우드 db, onlyoffice/documentserver까지 최종적으로 3개의 컨테이너가 필요하다. 넥클은 하던대로 설치한 후 ONLYOFFICE 앱 추가.

https://github.com/onlyoffice/Docker-DocumentServer

sudo docker run -i -t -d -p 8085:80 onlyoffice/documentserver

어차피 넥스트클라우드 공간을 빌려쓸건데 얘 볼륨을 따로 잡아줄 필요성을 못 느껴서 단순 실행. https 역방향 프록시 주소만 하나 빼 주자. onlyoffice 서버 설정이 https를 요구해.

https://api.onlyoffice.com/editors/signature/

For Linux - /etc/onlyoffice/documentserver/local.json.
For Windows - %ProgramFiles%\ONLYOFFICE\DocumentServer\config\local.json.

하지만 onlyoffice/documentserver 컨테이너 내부에서 시크릿 키는 알아올 필요가 있다. 해당 키를 cat해와서 https 주소와 함께 저장.

이제 대충 시놀에서 쓰던 맛이 난다. 넥클 기본앱들도 포토고 뭐고 다 꺼버리니까 속도도 그럭저럭 버틸만 하고. 이제 블로그만 넥클로 가져올까? 넥클 메모가 분류는 폴더라서 편하거든.
