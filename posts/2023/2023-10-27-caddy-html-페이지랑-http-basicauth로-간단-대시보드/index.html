[Caddy] HTML 페이지랑 HTTP basicauth로 간단 대시보드

미니멀리즘도 병이라 SFTPGo 덕에 파일브라우저/파일스태시를 걷어내고 보니 헤임달도 과해보인다. 외부에서는 직접 주소 입력, 내 기기에서는 북마크 쓰면서 뭐하러 따로 대시보드 컨테이너를? 근데 또 아예 없긴 좀 아쉬워. 간혹 주소 기억 애매하거나 다른 창에 타이핑하기 귀찮을 때 복사해오기 좋은데.

가만 생각해보니 그걸 Caddy가 다 채워줄 수 있잖아? 찾아보니 캐디 시큐리티 플러그인도 있어서 구글 등 외부 인증까지 도입할 수 있지만 과하다. 캐디 순정의 HTTP 베이직 인증만으로 충분해.

https://caddyserver.com/docs/caddyfile/directives/basicauth

https://josheli.com/knob/2021/02/24/single-sign-on-in-caddy-server-using-only-the-caddyfile-and-basic-authentication/

Caddyfile

서브.도메인.com {
	basicauth / {
		아이디 해쉬패스
	}
	root * ./경로
	encode gzip
	try_files ./경로/index.html
	file_server
}

caddy hash-password 명령으로 비번 해쉬를 만들어주고 그걸 아이디 옆에 붙여넣으면 된다. 캐디 진짜 너무 빠르고 간단해 좋아…..
