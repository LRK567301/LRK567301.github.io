VS Code 1.78.1 업데이트 후 UNC 경로 오류

https://github.com/microsoft/vscode/issues/182038

https://code.visualstudio.com/docs/setup/windows#_working-with-unc-paths

갑자기 멀쩡히 잘 쓰던 네트워크 폴더가 UNC host 액세스 어쩌구 오류를 띄워서 당황. 찾아보니 이번 업데이트 이후 허가된 호스트 네임만 접근할 수 있게 바뀌었나 보다. 설정에서 security.allowedUNCHosts를 검색해 접근할 경로의 호스트 네임 부분(\server-a\path라면 server-a)만 추가.
