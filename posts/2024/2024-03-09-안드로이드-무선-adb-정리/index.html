[안드로이드] 무선 ADB 정리

https://play.google.com/store/apps/details?id=com.github.standardadb&hl=ko&gl=US

스토어에서 ADB Shell - Debug Toolbox(ColorBox Technology) 설치하고 개발자 옵션-무선 디버깅을 상하 분할창으로 같이 열고 ‘페어링 코드로 기기 페어링’에서 표시되는 IP/포트/페어링 코드로 ADB Shell에서 접속.

카메라
adb shell settings put system csc_pref_camera_forced_shuttersound_key 0
GOS 끄기
adb shell pm disable-user --user 0 com.samsung.android.game.gos

이후는 setedit나 윈도우 유선 연결해서 하던 것과 동일.
