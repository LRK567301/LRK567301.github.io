B650M 바이오스 업데이트, WOL 설정

오랜만에 바이오스 확인했더니 1월에 베타 2개 나온 후 7D76vAB(2024-02-19)가 한달쯤 버티고 있길래 이번엔 괜찮지 않을까 도전. 7D76vA8(2023-11-08)에서 7D76vAB(2024-02-19)으로 점프했다. 이번엔 다행히 괜찮아. 바이오스 설정도 바로 먹고 부팅시간도 설정으로 줄어들었어.

https://kr.msi.com/support/technical_details/MB_Wake_On_LAN

이번엔 한참 노트북이랑 무선만 쓰느라 잊고 있었던 WOL도 켰다. 윈도우에서 빠른 시작부터 꺼 주고

시스템팬1 스마트 컨트롤 Enabled
시스템팬1 타입: PWM
MSI 드라이버 인스톨 유틸리티 끄기
CPU Game Boost 끄기
메모리 EXPO 프로필1(6000MHz)
Pbo: ADVANCED
Pbo Limits: Motherboard
Curve Optimizer 클럭별로 Nagative 30
Memory Context Restore: Enabled
Power Down Mode: Enabled

Memory Context Restore나 Power Down Mode는 검색하는게 더 빠르다.
