리브레오피스 vs 한컴오피스

한셀에서 3000행 정도 데이터에 자동 필터를 걸고 복사하는데 말도 안되게 버벅이길래 분노의 리브레오피스 전환으로 엑셀급 성능을 되찾았다. 설치파일 용량부터 꽤 차이나길래 이대로 정착하나 했는데 일주일쯤 굴려보니 아……..돈 내고 쓰는 SW가 확실히 낫네.

리브레오피스 25.2.4.3

  • 설치파일 300MB
  • 설치용량 700MB
  • winget 지원
  • 로고 스플래시는 금방 지나가는데 하얀 창만 먼저 띄우고 기다려서 로딩 더 느린 느낌
  • ods 파일 3초
  • xls 파일 2초
  • UI 엑셀과 확실히 다르지만 있을 건 다 있으니 상단 텍스트 메뉴에서 찾아야함
  • 기본글꼴 병신: 메뉴 한글도 데이터 한글도 윈도우와 친하지 않은 리눅스쪽의 그것이라 영어만 그럴싸하고 한국어도 일어도 안티앨리어싱 따위 없는 돋움 바탕 굴림
  • 뭐가 꼬였는지 xls는 괜찮은데 ods 열면 엔비디아 오버레이가 뜨면서 지싱크랑 충돌할때 화면 깜빡이는 증상 발생
  • 윈도우 다크테마라고 따라서 다크테마 맞춰주는데 본문 배경까지 블랙으로 덮어서 까만글자가 안보임
  • 기본으로 켜진 자동 맞춤법 검사도 끄자 거의 모든 단어를 빨간줄침

한컴오피스 2024 13.0.0.2151

  • 설치파일 1GB
  • 설치용량 2GB
  • 로고 스플래시에서 진짜 로딩 다 하고 데이터 바로 보여줌
  • ods 파일 4초 (리브레보다 확실히 늦음)
  • xls 파일 2초
  • UI 기본적으로 엑셀 닮아서 대충 그럴싸한데 훑다보면 찾는 기능 나오는데 오피스 호환 테마까지 있음
  • 윈도우 네이티브니 글꼴 깔끔, UI 및 아이콘도 비교불가로 매끈함
  • 보기 기본값이 선택된 셀 행과 열 모두 강조인건 의문 다른 애들과 달라서 헷갈림 꺼야함

여러모로 한컴이 압도적. 애먹였던 자동 필터도 1500행 정도에서 다시 했더니 잘 되더라. 한컴 쓰자.
