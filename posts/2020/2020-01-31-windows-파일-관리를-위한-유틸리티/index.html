[Windows] 파일 관리를 위한 유틸리티

Everything

빛. 안 그래도 빛 그 자체였는데 윈10 1909에서 검색창이 병신(특히 네트워크 폴더에서 몇번 검색하고 나면 뻗는)이 되면서 이 이상 높을 수 없었던 가치가 더 높이 뛰어올랐다. 나스 폴더들도 인덱스에 추가하고 윈도우 서비스로 등록해서 트레이에 자동실행까지 걸었다. 없이는 못 살겠다. 로컬 검색이 이렇게 어려운 일이야.

DoubleKiller 대신 WinMerge + SearchMyFiles

이거다 싶은 작품들은 책장 폴더로 복사해뒀지만 여전히 임시폴더에 혼재하던 책들을 싹 정리하려고 DoubleKiller를 꺼냈다. 중복 파일 정리에 특화된 프로그램답게 빠르고 간단하게 정리했다 싶었는데 Everything으로 검색하다 보니 남아있는 중복들이 보인다. CRC32 체크 때문에 살짝 달라진 파일은 빠졌나 싶어 다시 Run, 아…….유니코드………..낡아빠진 프로그램의 숙명. 그래도 네트워크 폴더는 지원해서 지금껏 끌고 왔는데 이제 이별할 때가 됐어.

텍스트 비교용으로 사용하던 WinMerge의 폴더 비교가 생각보다 더 편리하고 깔끔한 대안. DoubleKiller처럼 서브폴더 내 파일까지 중복만 간단히 걸러주진 않지만, 살짝 다른 파일 확인부터 중복 삭제 후의 폴더 상태까지 실시간으로 직관적으로 뿌려줘서 더 좋다. 사용법을 살짝 헤맸는데 언어를 한글로 바꾸고 비교 방법을 크기로 하니 원하던 속도가 나오더라. ‘빠른 내용’은 생각이랑 달랐어.

NirSoft의 SearchMyFiles도 추가. 안쓰는/구버전 유틸 폴더를 정리하다 찾았다. 얘도 중복 검색 기능이 있었네. CloneSpy보단 이런 목록 스타일이 훨씬 편해.

AwesomePhotoFinder + Picasa 3

프리웨어면서 퍼센테이지 비교, 네트워크 폴더까지 지원하는 툴이 흔치 않아서 AwesomePhotoFinder는 현상유지. 거기에 2016년에 3.9.141.259로 멈춘 피카사3를 추가했다. 이것저것 다 써봐도 피카사만큼 새 이미지 확인도 100% 동일 이미지 검출도 빠른 툴이 없더라. 윈10 사진 앱도 최근 사진을 뿌려주긴 하는데 로컬 파일만 빠르게 읽지 네트워크 파일은 하세월이야. 사진 앱을 직접 실행해서 한나절 열어둬야 새 파일을 읽어준다.

DropIt보다는 배치파일

트위터/픽시브/인스타/플리커 등에서 쏟아지는 이미지들을 받아 쉽게 분류하기 위한 방법. 다운로드 확장이 해당 계정명만 파일명 앞에 찍어주면 이후는 간단하다. DropIt의 넓은 활용범위도 좋지만 나는 파일 옮기기만 하면 되니 간단하게 배치파일로.

@echo off
chcp 65001
move /-Y "C:\Users\PC\Downloads\mery__S2_*.*" "\NAS\Documents\사진\mery__S2_"
move /-Y "C:\Users\PC\Downloads\umino_mkz*.*" "\NAS\Documents\사진\Comic\umino_mkz"
move /-Y "C:\Users\PC\Downloads\CUTiE_LILIA*.*" "\NAS\Documents\사진\Doll\CUTiE LILIA"

DropIt도 ini 안에 저장되지만 여러줄이라 일괄 편집하긴 귀찮은데 배치파일로 옮기기만 하는 건 간단하다. 필요할 때마다 한줄씩 추가해나가는 중. 물론 정말 자동화의 끝판왕은 DropIt의 감시 기능까지 쓰는 거겠지만 뭘 받았는지는 확인하고 정리하고 싶으니까. 물론 일단 다 옮겨버리도 피카사만 켜면 순식간에 새 사진들을 위에 띄워주긴 한다. 정렬은 꼬여있지만.
