webp 앨범아트도 입혀주는 youtube-dlG

GitHub - MrS0m30n3/youtube-dl-gui: A cross platform front-end GUI of the popular youtube-dl written in wxPython.

CLI를 고집하면서도 문서는 제대로 읽지도 않았더니 이런 편리함이 다 갖추어져 있는 줄도 몰랐다. 매번 다운로드만 시키고 태그랑 앨범아트는 수동으로 입히는 뻘짓을 했어.

youtubedl.exe -x --audio-format mp3 --add-metadata --embed-thumbnail

그런데 메타데이터까진 입혀주면서 –embed-thumbnail가 말썽이다. 썸네일을 빼면 webm에서 mp3로 변환, 태그까지 잘 입혀주는데 앨범아트만 넣으려고 하면 Conversion failed! 태그만 입혀진 mp3, 임시 temp.mp3, webp 앨범아트 파일만 덩그러니 남는다. webp 파일을 못 넣는듯.

[postprocessor/embedthumbnail] Fix Youtube’s WebP thumbnails by converting them to JPG by alexmerkel · Pull Request #25717 · ytdl-org/youtube-dl · GitHub

마침 최근 이슈였네. 그런데 youtube-dl-gui에선 앨범아트까지 입혀줘. 편한데다 기능적으로도 뛰어난 GUI 툴을 안 쓸 이유가 없어.
