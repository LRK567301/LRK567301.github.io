USB 3.0 포트 + USB 2.0 연장선 + 2.4Ghz 무선 리시버: 효과가 있다

USB C 허브에 HDMI랑 G304만 꽂아도 커서가 덜덜 떨리던 건 허브 탓이었다. 다기능한 씽크웨이 허브를 치우고 심플한 유그린 허브를 꽂았더니 HDMI/G304/K77/아젤G까지 다 꽂아도 꽤 잘 버텨. 하지만 그것도 의외로 잘 버틴다는거지 마우스까진 괜찮은데 K77은 게임 연속입력이 씹히더라. 오버워치하는데 이동하다 멈춰!

그래서 K77을 방출하려 했지만 인간이라는게 편함 앞에선 참 간사하기 짝이 없는 동물이라 그 잠깐 맛본 게 잊혀지질 않는다. 쿨마 되돌려놓고 LED뽕에 다시 차나 싶었는데, 마우스와 달리 키보드는 99% 고정위치니까 유선도 괜찮을 줄 알았는데 자꾸 신경이 쓰인다. 특히 북스캔한다고 책상 잠깐 치울 때 아……케이블 한줄이 이렇게 걸리적거렸나?

USB 장치를 최대한 멀리 떨어트리면 된다는 건 이미 몇번 들었지만 짧은 USB 2.0 허브로 연결했을 때는 별 차이가 없었고, 연장선을 길다랗게 뺄거면 유선 키보드랑 뭐 얼마나 다르겠나 했는데 다르다. 확실히 편해. 폰에 블투 DAC로 유선 이어폰 연결하는 거랑 비슷해. 좀 길게 뻗어도 중간 연결이 없다는 것만으로 완벽하진 않지만 부족하지 않은 자유가 찾아온다.

일부러 오버워치를 계속 돌렸다. 첫날은 잠깐 불안정한 듯한 모습도 보였는데 그 한번이었다. 스페이스바 통울림은 있지만 다른 키들은 쿨마보다 짤깍 도각거리는 좋은 키감을 보여주고, 디맥에도 부족함이 없어. LED고 나발이고 과연 편한 게 최고야.
