OpenSSH 접속 bat 파일을 Windows Terminal에서 실행시키기

2020-09-22

setting.json을 만져서 파워쉘이 아니라 CMD를 1번 기본값으로 바꾸고 나니 이제는 정말 putty를 버려도 되지 않을까? 한번 버렸다가 오라클 클라우드 때문에 돌아왔던건데 그 오라클 클라우드가 인스턴스 생성 과정에서 key 파일을 직접 주게 됐으니 게임 끝났지. 호환성도 떨어지는 ppk 귀찮다.

How to run a bat file in current windows terminal instead of open new cmd window? · Issue #5114 · microsoft/terminal · GitHub
How can I get batch files to run through the new Windows Terminal? - Stack Overflow
batch-file - 배치 스크립트의 상대 경로

wt cmd /c ssh -i "%~dp0/SSH키.key" 계정@IP주소

putty처럼 세션 관리 기능이 있는 것보단 불편하지만 단순 접속만은 배치파일로 더 빠르게 할 수 있다. key와 bat를 같은 폴더에 두고 실행. 나스 공유폴더 경로의 key를 바로 읽어오지 못하는(퍼미션 오류) 건 단점.
