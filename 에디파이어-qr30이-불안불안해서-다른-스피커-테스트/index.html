에디파이어 QR30이 불안불안해서 다른 스피커 테스트

2026-07-19

작년 9월쯤 들였던가? 아직 1년도 안 된 에디파이어 QR30의 왼쪽에서 며칠 전부터 지직거리는 잡음이 들리기 시작했다. 유튜브 음악 켜두니까 들리다 말다하길래

https://www.szynalski.com/tone-generator/

전원 콘센트 위치, USB 포트, 블루투스 소스 다 바꿔가면서 톤 제네레이터로 확인해보니 왼쪽만 100Hz 부근에서 작게 치지직. 에디파이어 블루투스도 별로 맘에 안들더라니 믿었던 유선까지 배신인가 허탈해하다 버릴 각오로 겉면 우퍼 부분을 꾹꾹 눌렀더니 괜찮아지긴 했는데 으으으으음…….오래 켜두면 소리 날때 지직거려서 전원을 껐다켜야 하는 문제도 처음부터 있었고(지금은 스마트플러그에 물려둠) 조명도 피크미터 같은 효과가 있는것도 아니라 기본값 어둡게만 쓰고 있어서 크게 의미없고…..이참에 다른 맛을 보자.

페블 2.0

다시 꺼내봐도 깡통. 뒷면 필름? 떼어내도 아무런 차이가 없다. 그냥 깡통. 모바일 스피커가 나아진 수준이야. 구형이라 AUX/USB A 둘이나 꽂아야 해서 번거롭지만 데스크탑 USB만으로 전원 유지되고 소리도 나는 스테레오 스피커로서의 편리함은 여전히 인정.

페블 V3

처음엔 2.0이나 얘나 그게 그거였는데 번갈아 들어보니 조금 커진만큼 확실히 낫다. 스피커는 체급이야. 2.0이 왼쪽에서 오른쪽으로 쭉 오르는 산이라면 V3는 중역이 꽤 찼고 저음도 살짝 늘었어. 게다가 USB C(+USB A 젠더) 하나만으로 출력되어 더 깔끔해졌다. 블루투스는 있으나 마나. XB100만도 못한데 이걸 왜 써.

게인모드도 바꿔봤는데 일반 데스크탑이나 폰 전원으로는 차이가 없다. 또 전원 / AUX 분리해야할 것 같은데 그러면 V3 쓰는 의미가 없어져.

크리에이티브 GS3

사운드바 치고 꽤 괜찮은데? 깡통 페블들과 달리 저음이 꽤 있다. 보컬이랑 고역도 선명한데 이건 에디파이어가 너무 부드럽고 낮고 따뜻해서 비교되는 것 같기도 하고 페블 따위랑은 급이 달라. 스테이징 확 넓어져서 보컬만 비교해도 훨씬 살이 붙었어. 하단으로 밝게 뿌려지는 조명효과도 예쁜데 피크미터까지 맛있고.

문제는 USB C 하나로 해결되지만 컴퓨터 전원 켤 때마다 얘도 켜줘야 한다는거. 페블처럼 전원버튼이 아날로그로 고정되지 않아 디지털 원클릭이 필요하고 볼륨도 윈도우 볼륨과 통합(USB 연결시 볼륨단위 4, AUX 연결시 볼륨 분리)되어 움직인다. 이 전원 문제는 크리에이티브의 다른 USB C 사운드바는 물론 레이저 레비아탄도 동일한 모양. 전원 / AUX 따로 꽂으면 되겠지만 USB C가 셀링포인트인데 그렇게는 못쓰는거나 다름없는게 아이러니.

그리고 음향효과 슈퍼와이드는 쓰레기. 소리 진짜 이상하게 왜곡시킨다.

QR30

역시 스피커는 체급. 크리에이티브 한바퀴 돌고 나니까 저음 너무 많고 보컬도 먹먹하지 않나(오버워치하는데 총소리 선예도 차이가 ㅋㅋ) 싶지만 쫙 깔리는 중저음에서 나오는 스테이징이랑 무게감이 달라.

당장 바꾼다면 전원 처리는 아쉽지만 GS3. QR30 크기도 살짝 부담스러운데 체급 따라 페블을 X까지 올라가면 가격도 부피도 에디파이어 못지 않아진다. 오랜만에 써보는 사운드바가 역시 깔끔하고 조명도 예뻐.

에디파이어 M60

QR30 옆그레이드. 비슷한 크기라 갈아타는 의미가 별로 없다.

에디파이어 R20BT

USB로만 연결되는 2채널 초소형 직구 3만원대. 페블 갈거면 얘다. 근데 전원이 확실치 않아. USB 꽂자마자 켜지는 것 같은데 출력까지 바로 되는지는 모르겠어.

에디파이어 QS30

얘네 진짜 음향 안하는게 없구나 USB 사운드바도 있어. 하지만 QR30 하위호환으로 생기다 만 디자인과 블루투스 라인업과 마찬가지로 1.5초 온오프로 표기된 전원이 매우 불길하다.

크리에이티브 GS5

AC 전원으로 빠지면서 오히려 전원 고민할 일이 없어졌다. GS3보다 크기도 10cm 긴데 그게 오히려 27인치~32인치 모니터 아래엔 딱 맞을 느낌. 리뷰 찾아보니 GS3 고역이 밝고 GS5는 저음 많고 차분하다는 거 보니 에디파이어 느낌도 날듯? 리모컨도 있고 앱이랑 EQ도 지원하고 할인하면 노려볼만해.
