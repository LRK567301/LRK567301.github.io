OpenVPN 도커로 교체

15159-20-21T29:28:00+09:00

기존 사용하던 네이티브 스크립트는 얘. WireGuard가 그렇게 좋다니 갈아타는 것도 고려했지만 드디어 OpenVPN 지원을 시작한 아이피타임 공유기나 시놀로지 VPN 연결도 L2TP/OpenVPN인걸 보니 아직은 OpenVPN.

https://hub.docker.com/r/kylemanna/openvpn

OVPN_DATA="적당한이름"
docker volume create --name $OVPN_DATA
docker run -v $OVPN_DATA:/etc/openvpn --rm kylemanna/openvpn ovpn_genconfig -u udp://아이피
docker run -v $OVPN_DATA:/etc/openvpn --rm -it kylemanna/openvpn ovpn_initpki
docker run -v $OVPN_DATA:/etc/openvpn -d -p 1194:1194/udp --cap-add=NET_ADMIN kylemanna/openvpn
docker run -v $OVPN_DATA:/etc/openvpn --rm -it kylemanna/openvpn easyrsa build-client-full 적당한이름 nopass
docker run -v $OVPN_DATA:/etc/openvpn --rm kylemanna/openvpn ovpn_getclient 적당한이름 > 적당한이름.ovpn

역시 도커가 훨씬 간단하다. 볼륨 마운트 필요없나 쓸데없이 고민했는데 하라는 대로만 하면 현재 경로에 ovpn 파일 빼주더라.
