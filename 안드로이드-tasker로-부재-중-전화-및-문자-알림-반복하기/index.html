포코 F5 AOD 10초? 없으면 안 쓰면 되지 했는데 게임기가 아니라 서브폰으로 실사용에 들어가니까 치명적인 문제였다. 첫 알림 놓치면 폰 집어들기 전까지 알 수가 없어. AOD도 LED도 없으니 답이 없다. EU롬 올리면 항상 켜기도 된다는데 미맥스로 순정이 최고임은 몇번을 확인했고 언락하려면 168시간인가 대기시간도 필요하고….귀찮다. 써드파티로 비벼보자.

돌고 돌아 태스커였다. 스토어에 그럴싸해보이는 앱들 깔아봤는데 쓰레기야. 근본은 역시 근본. 빅스비 루틴의 편리함에 길들여져서 오랜만에 보는 태스커 너무 복잡하고 어려워보여서 헤맸는데 사람 더 정신없게 만드는 한국어 지원 치우고 영어로 차근차근 보니까 간단하게 짤 방법이 보인다. 인터넷에서 찾은 프로필들은 2023년에 안드로이드13에서 쓰기엔 다 너무 옛날 거야.

PROFILES
State-Unread Text Any + Time-Every 2m
Event-Missed Call C:ANY + Time-Every 2m

TASKS
Vibrate-Time 500, Turn On-Block Time 500

프로필 두개에 태스크를 하나씩 할당해주면 2분마다 화면이 켜지면서 진동이 울린다. AOD만 되면 알림 있을 때만 AOD 켜기가 최고겠지만 별 수 없지.
